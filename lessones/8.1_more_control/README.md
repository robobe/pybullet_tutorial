# More control

## Control

### Mass-Spring-Damper
This is the most important simple control system.

!!! tip ""
    This is the BEST system for learning control.
    
$$m ẍ + c ẋ + kx = F$$

#### state space
```
x1 = position
x2 = velocity

x1_dot = x2
x2_dot = (-k/m)x1 - (c/m)x2 + (1/m)F
```

#### Usage
- Apply force
- Add PID
- Use LQR
- Use RL
